# NumOpt #

Set of codes and exercises for numerical optimization learning

### What is this repository for? ###

* Education regarding numerical optimization techniques
* Solving the problems in Martins' book 


### How do I get set up? ###

* Git clone the present folder in your system

* Create a new python environment:
```bash
conda create -n NumOpt python=3.12
```

* Activate the new environment:
```bash
conda activate NumOpt
```

* Navigate to the package and and install it on your machine:
```bash
pip install -e .
```
